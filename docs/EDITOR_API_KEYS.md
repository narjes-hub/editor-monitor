## How to Use

1. Give each editor their unique API key
2. They go to: `https://your-vercel-url.vercel.app`
3. Enter their API key
4. Select tool (Veo 3, MakeUGC, ElevenLabs, Higgsfield)
5. Enter their script/prompt
6. Submit

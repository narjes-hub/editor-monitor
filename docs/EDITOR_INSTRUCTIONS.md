# Editor Instructions - W16 Script Portal

Welcome! This guide explains how to use the W16 Script Portal to submit your scripts for approval before creating content.

---

## Why Use the Portal?

All scripts must be submitted through the portal **before** using any AI tool (Veo 3, MakeUGC, ElevenLabs, Higgsfield). This helps us:

- Track content creation across the team
- Ensure all content aligns with our brands
- Maintain quality standards

---

## Your API Key

You have been assigned a unique API key. **Keep it private** - it identifies all your submissions.

Your key looks like: `ap_yourname_xxxxxxxxxxxx` or `le_yourname_xxxxxxxxxxxx`

- `ap_` = Aurapets brand
- `le_` = LuminEye brand

## Important Rules

1. **Always submit first** - Never use AI tools without portal approval
2. **Use your own key** - Never share or use someone else's API key
3. **Complete scripts only** - Submit the full script, not just notes
4. **One script per submission** - Submit each script separately

---

## FAQ

**Q: What if I forget my API key?**
A: Contact your supervisor to retrieve it.

**Q: Can I submit multiple scripts at once?**
A: Submit one at a time. After approval, click "Submit Another Script."

**Q: What if I'm working on both brands?**
A: You'll have separate API keys for each brand. Use the correct key for each project.

**Q: How long does approval take?**
A: Instant - usually 2-3 seconds.

**Q: What if the portal is down?**
A: Contact your supervisor. Do not proceed without approval.
